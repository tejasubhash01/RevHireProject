About RevHire

Where talent meets opportunity – seamlessly.

Application Overview

RevHire is a full-stack monolithic web application that serves as a job portal connecting job seekers with employers. Job seekers can create profiles, build and upload resumes, search for jobs using advanced filters, and track their applications. Employers can post job openings, manage applications, and shortlist or reject candidates. The application features a responsive web interface, role-based access control, and real-time notifications for an enhanced user experience.

Core Functional Requirements

As a Job Seeker

  • Register and create an account with personal details
  • Login using email and password
  • Create and manage textual resume with structured sections
  • Upload formatted resume (PDF/DOCX, max 2MB)
  • Search jobs using advanced filters
  • Apply for jobs with one-click using saved resume
  • View all applications with status
  • Withdraw applications with confirmation
  • Manage profile (education, experience, skills, certifications)
  • Receive in-app notifications
  • Save jobs to favourites
  • View detailed job descriptions and company profiles

As an Employer

  • Register company and create account
  • Login using registered credentials
  • Create job postings with comprehensive details
  • Manage job postings (edit, close/reopen, delete, mark as filled)
  • View applicant details including profile and resume
  • Shortlist or reject applications with bulk actions
  • Search and filter applicants by experience, skills, education, date, status
  • Manage company profile and update information
  • View dashboard with statistics (jobs, applications, pending reviews)
  • Add notes to applications for internal tracking
